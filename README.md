# World Cup 2026 Prediction Contest

Full-stack web application for managing a private FIFA World Cup prediction competition.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL + Auth + RLS)

## Setup
1. `npm install`
2. Copy `.env.example` to `.env.local` and fill in Supabase credentials
3. Run SQL migrations in `supabase/migrations/`
4. `npm run dev`
